# Ocean Science Solutions Website

Static website for Ocean Science Solutions, LLC.

## GitHub Pages

This repo is formatted for branch-based GitHub Pages deployment from the `docs/` directory.

In the repository settings, use:

- **Settings > Pages > Source:** Deploy from a branch
- **Branch:** `master` or `main`, whichever branch you publish from
- **Folder:** `/docs`

The custom domain is configured in `docs/CNAME`:

```text
oceanscientific.net
```
